# 终极震荡指标策略 (Ultimate Oscillator Strategy)

## 策略简介

终极震荡指标（Ultimate Oscillator，UO）是由Larry Williams开发的技术分析工具，通过整合三个不同时间周期（7日、14日、28日）的数据来减少虚假信号，提供更可靠的超买超卖判断。

## 策略原理

### 指标构成

**终极震荡指标 (Ultimate Oscillator)**
- 计算方法：基于三个不同周期的平均真实波幅（ATR）
- 三个周期：7日（短期）、14日（中期）、28日（长期）
- 加权组合：三个周期按特定权重组合
- 取值范围：0-100

指标解读：
- 70以上：超买区域
- 30以下：超卖区域
- 背离信号：价格创新高但UO不创新高（看跌背离）

### 交易逻辑

#### 入场条件（做多）
- UO < 30（进入超卖区域）
- 表示短期超卖可能反弹

#### 出场条件
- UO > 70（进入超买区域）
- 表示短期超买可能回调

### 策略特点

- 仅做多（Long-only）
- 多时间框架整合
- 减少虚假信号
- 适合均值回归

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| p1 | 7 | 短期周期 |
| p2 | 14 | 中期周期 |
| p3 | 28 | 长期周期 |
| oversold | 30 | 超卖阈值 |
| overbought | 70 | 超买阈值 |

## 适用场景

### 最佳适用
- 震荡行情
- 需要减少虚假信号的交易
- 多周期分析方法

### 不适用场景
- 强趋势行情（可能过早离场）
- 持续创新高或新低的市场
- 需要快速反应的策略

## 风险提示

1. **趋势错过**：强趋势中可能过早退出
2. **信号较少**：严格条件导致交易频率较低
3. **滞后性**：多周期整合带来一定滞后
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **阈值调整**：根据市场特性调整超买超卖阈值
2. **背离交易**：结合价格与指标的背离信号
3. **止损机制**：添加ATR止损保护
4. **仓位管理**：根据UO位置动态调整仓位
5. **趋势过滤**：结合ADX判断趋势强度
