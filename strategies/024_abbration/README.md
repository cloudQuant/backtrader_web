# 布林带突破策略 (Abbration Bollinger Band Breakout Strategy)

## 策略简介

布林带突破策略是一种基于布林带指标的趋势跟踪策略。当价格突破布林带上轨时产生做多信号，突破下轨时产生做空信号，当价格回归中轨时平仓离场。

## 策略原理

### 指标说明

- **布林带 (Bollinger Bands)**: 由上轨、中轨（简单移动平均线）和下轨组成，用于判断价格的波动范围和趋势方向。

### 交易规则

**开仓条件：**
- 做多：价格从下方突破布林带上轨（前一根K线收盘价<上轨，当前K线收盘价>上轨）
- 做空：价格从上方跌破布林带下轨（前一根K线收盘价>下轨，当前K线收盘价<下轨）

**平仓条件：**
- 多头平仓：价格从上方跌破中轨
- 空头平仓：价格从下方突破中轨

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| boll_period | 200 | 布林带周期 |
| boll_mult | 2 | 布林带标准差倍数 |

## 适用场景

- 价格波动较大的股票或期货品种
- 趋势明显的市场环境
- 震荡突破后的趋势行情

## 风险提示

1. 震荡市场中容易产生假突破
2. 突破失败可能造成连续亏损
3. 建议结合其他指标过滤信号

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：16笔
- 胜率：56.25%
- 最终资金：423,916.71元（初始100,000元）
- 年化收益率：6.95%
- 最大回撤：46.52%
