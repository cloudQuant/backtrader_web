# 动量策略 (Momentum Strategy)

## 策略简介

动量策略是一种基于价格动量指标的趋势跟踪策略，通过测量价格变化的速度来识别趋势方向的变化。

## 策略原理

### 指标构成

**动量指标 (Momentum)**
- 计算公式：当前价格 - N期前的价格
- 默认周期：20日
- 正值表示价格上涨动量，负值表示价格下跌动量

### 交易逻辑

#### 入场条件（做多）
- 动量指标由负值穿越零轴变为正值
- 前一根K线动量 <= 0 且当前K线动量 > 0

#### 出场条件
- 动量指标由正值穿越零轴变为负值
- 前一根K线动量 > 0 且当前K线动量 <= 0

### 策略特点

- 仅做多（Long-only）
- 趋势跟踪策略
- 基于零轴交叉产生信号
- 简单直观的交易逻辑

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| period | 20 | 动量指标计算周期 |

## 适用场景

### 最佳适用
- 趋势明显的市场
- 价格持续性较强的行情
- 波动率适中的市场

### 不适用场景
- 震荡横盘市场
- 频繁反转的市场
- 低波动率市场

## 回测结果

使用Oracle (ORCL) 2010-2014年数据回测：

| 指标 | 数值 |
|------|------|
| 初始资金 | $100,000 |
| 最终价值 | $100,040.76 |
| 年化收益率 | 0.008% |
| 夏普比率 | 0.26 |
| 最大回撤 | 8.33% |

## 风险提示

1. **信号滞后**：动量指标基于历史数据，存在天然滞后性
2. **震荡亏损**：在横盘震荡中容易产生连续亏损
3. **无止损机制**：策略未设置止损，存在较大回撤风险
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数优化**：调整周期以适应不同市场特征
2. **趋势过滤**：结合ADX等指标判断趋势强度
3. **止损机制**：添加ATR或百分比止损
4. **仓位管理**：根据动量强度动态调整仓位
5. **多重确认**：增加成交量或其他指标确认
