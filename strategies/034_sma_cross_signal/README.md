# SMA交叉信号策略 (SMA Cross Signal Strategy)

## 策略简介

SMA交叉信号策略是最经典的移动平均线交叉策略。当短期均线上穿长期均线时买入（金叉），下穿时卖出（死叉）。

## 策略原理

### 指标说明

- **短期SMA**: 快速响应价格变化
- **长期SMA**: 反映长期趋势
- **CrossOver指标**: 检测均线交叉信号

### 交易规则

**开仓条件：**
- 做多：短期SMA上穿长期SMA（金叉，crossover > 0）

**平仓条件：**
- 平仓：短期SMA下穿长期SMA（死叉，crossover < 0）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| sma1 | 10 | 短期均线周期 |
| sma2 | 20 | 长期均线周期 |

## 适用场景

- 趋势明确的市场
- 中长期趋势跟踪
- 适合各类交易品种

## 风险提示

1. 震荡市中频繁交叉造成亏损
2. 信号存在滞后性
3. 可考虑添加其他指标过滤

## 常见优化方向

1. 使用EMA代替SMA以更快响应
2. 添加成交量过滤
3. 添加趋势强度指标（如ADX）过滤
4. 优化均线周期参数

## 回测表现

基于2005-2006年数据回测：
- 总交易次数：14笔
- 胜率：42.86%
- 最终资金：105,288.60元（初始100,000元）
- 年化收益率：2.57%
- 最大回撤：3.14%
- 夏普比率：1.67
