# 三重指数平滑平均线策略 (TRIX Strategy)

## 策略简介

三重指数平滑平均线（Triple Exponential Average，TRIX）是由Jack Hutson开发的技术指标，通过对价格进行三次指数平滑处理，有效过滤掉短周期波动，突出显示潜在的趋势方向。

## 策略原理

### 指标构成

**TRIX指标**
- 计算方法：
  1. 计算价格的单指数移动平均
  2. 对结果计算双指数移动平均
  3. 对结果计算三指数移动平均
  4. 计算三重EMA的变化率
- 默认周期：15日

指标解读：
- TRIX > 0：上升趋势
- TRIX < 0：下降趋势
- TRIX穿越零轴：趋势反转信号

### 交易逻辑

#### 入场条件（做多）
- TRIX从零轴下方穿越到上方
- 表示动量转为正值

#### 出场条件
- TRIX从零轴上方穿越到下方
- 表示动量转为负值

### 策略特点

- 仅做多（Long-only）
- 有效过滤短期噪声
- 适合识别趋势变化
- 滞后性适中

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| period | 15 | TRIX计算周期 |

## 适用场景

### 最佳适用
- 趋势明显的市场
- 需要过滤短期波动的交易
- 波动率较高的市场

### 不适用场景
- 持续横盘的窄幅震荡
- 极端事件驱动的市场
- 需要快速反应的策略

## 回测结果

| 初始资金 | $100,000 |

## 风险提示

1. **信号滞后**：三次平滑导致信号有一定的滞后性
2. **错过起点**：可能错过趋势的早期阶段
3. **震荡亏损**：在无趋势市场可能产生连续亏损
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **周期调整**：根据市场特征调整TRIX周期
2. **趋势过滤**：结合ADX判断趋势强度
3. **止损机制**：添加ATR止损保护
4. **仓位管理**：根据TRIX绝对值大小调整仓位
5. **多重确认**：增加成交量或其他指标确认
