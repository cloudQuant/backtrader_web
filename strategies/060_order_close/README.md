# 收盘价订单执行策略 (Order Close Strategy)

## 策略简介

本策略演示了如何在Backtrader中使用`bt.Order.Close`执行类型在每根K线的收盘价执行订单。策略使用简单的SMA交叉信号生成交易指令。

## 策略原理

1. **技术指标**：使用15周期简单移动平均线(SMA)
2. **买入信号**：当收盘价向上穿越SMA时买入
3. **卖出信号**：当收盘价向下穿越SMA时平仓
4. **订单执行**：所有订单使用`exectype=bt.Order.Close`在收盘价执行

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period | 15 | SMA周期 |

## 适用场景

- 需要在收盘价执行交易的场景
- 演示Backtrader的订单执行类型
- 简单的趋势跟踪策略测试

## 风险提示

- 本策略仅用于演示目的
- 在实际交易中，收盘价执行可能与回测结果存在差异
- 策略未考虑止损、止盈等风险控制措施
