# 布林带均值回归策略 (Bollinger Bands Mean Reversion Strategy)

## 策略简介

布林带是一种常用的技术分析工具，本策略基于布林带实现均值回归交易，在价格超出正常波动范围时进行反向交易。

## 策略原理

1. **布林带构成**：
   - 中轨：20周期SMA
   - 上轨：中轨 + 2倍标准差
   - 下轨：中轨 - 2倍标准差

2. **交易逻辑**：
   - **买入信号1**：价格跌破下轨(标记超卖)，回升至中轨时买入
   - **买入信号2**：价格直接突破上轨(动量买入)
   - **卖出信号**：价格从中轨回落

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| bbands_period | 20 | 布林带周期 |
| devfactor | 2.0 | 标准差倍数 |

## 适用场景

- 震荡市和均值回归行情
- 波动率相对稳定的市场
- 逆势抄底交易

## 风险提示

- 趋势行情中容易被套
- 突破行情中假信号较多
- 需要配合趋势判断指标

## 策略优势

- 能够识别超买超卖区域
- 结合了均值回归和动量策略
- 布林带宽度可反映波动率
