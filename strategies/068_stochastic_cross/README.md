# 随机指标交叉策略 (Stochastic Cross Strategy)

## 策略简介

本策略结合移动平均线趋势判断和随机指标的超买超卖信号，在趋势回调时寻找入场机会。

## 策略原理

1. **趋势判断**：
   - 上升趋势：短期MA > 长期MA
   - 下降趋势：短期MA < 长期MA

2. **交易信号**：
   - **买入**：上升趋势 + 随机指标%K < 20 (超卖)
   - **卖出**：下降趋势 + 随机指标%K > 80 (超买)

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| ma1 | 14 | 短期均线周期 |
| ma2 | 30 | 长期均线周期 |
| stoch_period | 14 | 随机指标周期 |
| oversold | 20 | 超卖阈值 |
| overbought | 80 | 超买阈值 |

## 适用场景

- 趋势回调后的反弹交易
- 震荡市中的高抛低吸
- 日内或波段交易

## 风险提示

- 趋势反转时可能连续亏损
- 随机指标在强趋势中可能持续钝化
- 需要配合止损策略

## 策略优势

- 结合趋势和振荡器指标
- 在趋势回调时入场，风险收益比较高
- 参数相对固定，适应性强
