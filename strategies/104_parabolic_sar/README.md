# 抛物线SAR策略 (Parabolic SAR Strategy)

## 策略简介

抛物线转向（Parabolic SAR）指标是由J. Welles Wilder Jr.开发的技术分析工具，用于确定趋势方向和潜在的反转点。SAR代表"Stop And Reverse"（止损并反向），该指标在强趋势市场中特别有效。

## 策略原理

### 指标构成

**抛物线转向指标 (Parabolic SAR)**
- 计算公式：复杂的加速因子算法
- 初始加速因子：0.02
- 最大加速因子：0.2
- 指标形态：价格上方或下方的一系列点

指标解读：
- 点在价格下方：上升趋势（做多）
- 点在价格上方：下降趋势（做空或观望）
- 点位切换：趋势反转信号

### 交易逻辑

#### 入场条件（做多）
- 收盘价向上穿越SAR线
- SAR从价格上方切换到下方
- 表明趋势可能转为上升

#### 出场条件
- 收盘价向下穿越SAR线
- SAR从价格下方切换到上方
- 表明趋势可能反转

### 策略特点

- 仅做多（Long-only）
- 趋势跟踪系统
- SAR同时作为动态止损位
- 加速机制随趋势增强

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| af | 0.02 | 初始加速因子 |
| afmax | 0.2 | 最大加速因子 |

## 适用场景

### 最佳适用
- 明显趋势行情
- 持续性强的趋势
- 波动率适中但趋势明确的市场

### 不适用场景
- 震荡横盘市场（频繁信号）
- 假突破频繁的市场
- 跳空剧烈的市场

## 回测结果

使用Oracle (ORCL) 2010-2014年数据回测：

| 指标 | 数值 |
|------|------|
| 初始资金 | $100,000 |
| 最终价值 | $100,044.47 |
| 年化收益率 | 0.009% |
| 夏普比率 | 0.16 |
| 最大回撤 | 14.47% |

## 风险提示

1. **震荡市亏损**：在横盘震荡中频繁进出导致连续亏损
2. **趋势误判**：趋势启动时可能产生错误信号
3. **加速机制**：加速因子过大可能导致止损过紧
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数调整**：根据市场特征调整加速因子
2. **趋势过滤**：加入ADX等指标判断趋势强度
3. **止损保护**：添加额外止损保护
4. **仓位管理**：根据趋势强度动态调整仓位
5. **多重确认**：结合成交量或其他指标确认
