# 海龟交易策略 (Turtle Trading Strategy)

## 策略简介

海龟交易策略是著名的趋势跟踪策略，由Richard Dennis和William Eckhardt在1983年开发。该策略使用移动平均判断市场多空状态，价格突破时进场，设置移动止损保护利润。

## 策略原理

### 指标说明

- **简单移动平均线 (SMA)**: 判断市场趋势（多空状态）
- **价格变化率**: 检测突破信号

### 交易规则

**开仓条件（需同时满足）：**
- 价格 > 长期均线（牛市状态）
- 价格变化率 > 阈值（突破信号）

**平仓条件：**
- 价格跌破长期均线（趋势反转）

**止损规则：**
- 使用移动止损（StopTrail）保护利润
- 止损距离为入场价的trailpercent百分比

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| maperiod | 15 | 短期均线周期 |
| breakout_period_days | 20 | 突破周期 |
| price_rate_of_change_perc | 0.1 | 价格变化率阈值（10%） |
| regime_filter_ma_period | 200 | 趋势过滤均线周期 |
| trailing_stop_loss_perc | 0.1 | 移动止损百分比（10%） |

## 适用场景

- 趋势明显的市场
- 长期趋势跟踪
- 适合期货和外汇品种

## 风险提示

1. 在震荡市场中表现不佳
2. 需要承受较大回撤
3. 移动止损可能在回调中过早离场

## 策略背景

海龟交易实验是著名的交易实验，证明了交易方法可以被教授。原版海龟策略使用：
- 20日突破入场
- 10日低点离场
- 2%的资金管理规则

本实现是简化版本，使用价格变化率和长期均线过滤。

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：46笔
- 胜率：36.96%
- 最终资金：100,008.96元（初始100,000元）
- 年化收益率：接近0%
- 最大回撤：2.45%
