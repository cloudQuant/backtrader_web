# 抄底策略 (Buy Dip Strategy)

## 策略简介

抄底策略是一个简单直观的均值回归策略，通过识别短期价格下跌（连续两根阴线）来寻找买入机会，并在固定持有期后卖出。

## 策略原理

### 核心思想
- 连续两根阴线通常表示短期超卖
- 短期下跌后有反弹概率
- 固定持有期避免过度持仓

### 交易规则
**买入条件：**
- close[0] < close[-1] 且 close[-1] < close[-2]（连续两根阴线）

**卖出条件：**
- 持有N根K线后自动卖出（默认5根）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| hold_bars | 5 | 持有K线数量 |

## 适用场景

- 震荡市或轻微下跌后的反弹
- 流动性好的品种
- 短期交易

## 风险提示

1. **持续下跌风险**：在持续下跌趋势中抄底可能被套
2. **固定持仓局限**：不考虑市场状况的固定持仓可能错失良机或过早退出
3. **无止损机制**：策略没有止损，亏损可能扩大
4. **交易成本**：频繁交易会增加手续费成本

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1257
- 夏普比率：1.030
- 年化收益率：0.018%
- 最大回撤：8.38%
- 最终资金：100,087.43
