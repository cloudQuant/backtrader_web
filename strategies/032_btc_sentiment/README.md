# 比特币情绪指标策略 (BTC Sentiment Strategy)

## 策略简介

比特币情绪指标策略基于Google Trends情绪数据和布林带的比特币交易策略。情绪超过布林带上轨时做多，低于下轨时做空，回归中性区域时平仓。

## 策略原理

### 数据来源

- **BTC价格数据**: 来自Yahoo Finance的比特币历史价格
- **Google Trends情绪数据**: 搜索量数据反映市场关注度

### 交易规则

**开仓条件：**
- 做多：Google情绪指标 > 布林带上轨
- 做空：Google情绪指标 < 布林带下轨

**平仓条件：**
- 情绪指标回归中性区域（介于上下轨之间）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period | 10 | 布林带周期 |
| devfactor | 1 | 标准差倍数 |

## 适用场景

- 加密货币市场
- 情绪驱动的交易品种
- 需要情绪数据的特殊策略

## 风险提示

1. 需要获取Google Trends数据
2. 情绪指标可能与价格走势背离
3. 加密货币市场波动极大

## 回测表现

基于BTCUSD 2018-2020年周线数据回测：
- 初始资金：10,000美元
- 总交易次数：16笔
- 胜率：50%
- 最终资金：15,301.43美元
- 年化收益率：23.70%
- 最大回撤：17.49%
- 夏普比率：0.80

## 数据要求

- BTCUSD价格数据：`BTCUSD_Weekly.csv`
- Google Trends数据：`BTC_Gtrends.csv`
