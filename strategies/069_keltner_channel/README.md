# 肯特纳通道策略 (Keltner Channel Strategy)

## 策略简介

肯特纳通道是由Chester Keltner开发的趋势跟踪指标，与布林带类似，但使用ATR(平均真实波幅)而非标准差来计算通道宽度。

## 策略原理

1. **通道计算**：
   - 中轨：20周期EMA
   - 上轨：中轨 + 2倍ATR
   - 下轨：中轨 - 2倍ATR

2. **交易逻辑**：
   - **买入**：收盘价突破上轨
   - **卖出**：收盘价回落至中轨下方

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| period | 20 | EMA周期 |
| atr_mult | 2.0 | ATR倍数 |

## 适用场景

- 趋势明显的市场
- 突破交易策略
- 与布林带类似的场景

## 与布林带的区别

- 肯特纳通道使用ATR，布林带使用标准差
- 肯特纳通道对波动的反应更平滑
- 肯特纳通道更适合趋势跟踪

## 风险提示

- 震荡市场中假突破较多
- 突破后可能快速回调
- 建议结合其他指标过滤信号

## 策略优势

- 基于ATR，更能反映市场波动
- 通道宽度动态调整
- 适合趋势跟踪
