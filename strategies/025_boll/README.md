# 布林带趋势跟踪策略 (Bollinger Band Trend-Following Strategy)

## 策略简介

布林带趋势跟踪策略是一种基于布林带突破的趋势跟踪策略。当价格连续突破布林带上轨时做多，连续跌破下轨时做空，并设置止损保护。

## 策略原理

### 指标说明

- **布林带 (Bollinger Bands)**: 用于判断价格波动范围和趋势方向

### 交易规则

**开仓条件：**
- 做多：收盘价连续两根K线高于上轨
- 做空：收盘价连续两根K线低于下轨

**平仓条件：**
- 多头平仓：收盘价跌破中轨 或 触发止损
- 空头平仓：收盘价突破中轨 或 触发止损

**止损规则：**
- 多头止损：入场价 - 当前价 > price_diff
- 空头止损：当前价 - 入场价 > price_diff

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period_boll | 245 | 布林带周期 |
| price_diff | 0.5 | 止损价格差 |

## 适用场景

- 趋势明显的市场环境
- 波动率适中的品种
- 能够承受一定回撤的趋势跟踪者

## 风险提示

1. 连续突破要求可能错过部分机会
2. 震荡市中止损可能频繁触发
3. 参数需要根据不同品种调整

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：20笔
- 胜率：30.00%
- 最终资金：325,630.39元（初始100,000元）
- 年化收益率：5.65%
- 最大回撤：45.74%
