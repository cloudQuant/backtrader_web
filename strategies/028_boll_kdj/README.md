# 布林带KDJ组合策略 (Bollinger Bands + KDJ Strategy)

## 策略简介

布林带KDJ组合策略采用双重确认机制，布林带突破提供趋势信号，KDJ金叉死叉提供入场时机确认。

## 策略原理

### 指标说明

- **布林带 (Bollinger Bands)**: 判断价格突破方向
- **KDJ指标**: 随机指标，用于判断超买超卖和金叉死叉信号

### 交易规则

**开仓条件（需双重确认）：**

做多：
- 布林带信号：价格从上方跌破下轨
- KDJ信号：J线上穿D线（金叉），且KDJ三线均<=25（低位金叉）

做空：
- 布林带信号：价格从下方突破上轨
- KDJ信号：J线下穿D线（死叉），且KDJ三线均>=75（高位死叉）

**平仓条件：**
- 多头平仓：触发止损 或 出现做空信号组合
- 空头平仓：触发止损 或 出现做多信号组合

**止损规则：**
- 多头止损：入场价 - 当前价 > price_diff
- 空头止损：当前价 - 入场价 > price_diff

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| boll_period | 53 | 布林带周期 |
| boll_mult | 2 | 布林带标准差倍数 |
| kdj_period | 9 | KDJ周期 |
| kdj_ma1 | 3 | KDJ快速均线周期 |
| kdj_ma2 | 3 | KDJ慢速均线周期 |
| price_diff | 0.5 | 止损价格差 |

## 适用场景

- 趋势明确的市场
- 需要双重确认的稳健交易
- 适合短线波段操作

## 风险提示

1. 双重确认可能错过部分机会
2. KDJ在震荡市中容易产生假信号
3. 需要针对不同品种优化参数

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：66笔
- 胜率：未明确
- 最终资金：75,609.19元（初始100,000元）
- 年化收益率：-1.29%（亏损）
- 最大回撤：66.05%
