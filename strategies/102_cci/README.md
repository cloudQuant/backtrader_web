# 商品通道指标策略 (CCI Strategy)

## 策略简介

商品通道指标（Commodity Channel Index，CCI）是由Donald Lambert开发的动量震荡指标，用于识别市场的超买和超卖状态。虽然最初是为商品市场设计，但现已广泛应用于各类金融市场。

## 策略原理

### 指标构成

**商品通道指标 (CCI)**
- 计算公式：CCI = (TP - SMA(TP)) / (0.015 × MD)
- 其中TP = (最高价 + 最低价 + 收盘价) / 3
- MD = 平均偏差
- 默认周期：20日

指标解读：
- +100以上：超买区域
- -100以下：超卖区域

### 交易逻辑

#### 入场条件（做多）
- CCI从-100以下向上穿越-100
- 表示从超卖区域开始回升

#### 出场条件
- CCI从+100以上向下穿越+100
- 表示从超买区域开始回落

### 策略特点

- 仅做多（Long-only）
- 基于阈值穿越产生信号
- CCI无界特性使其能捕捉极端行情
- 适用于趋势和震荡市场

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| period | 20 | CCI计算周期 |
| oversold | -100 | 超卖阈值 |
| overbought | 100 | 超买阈值 |

## 适用场景

### 最佳适用
- 具有明确趋势的市场
- 周期性波动的商品市场
- 波动率适中的市场

### 不适用场景
- 持续横盘的窄幅震荡
- 跳空频繁的市场
- 极端事件驱动的市场

## 回测结果

使用Oracle (ORCL) 2010-2014年数据回测：

| 指标 | 数值 |
|------|------|
| 初始资金 | $100,000 |
| 最终价值 | $100,196.64 |
| 年化收益率 | 0.039% |
| 夏普比率 | 1.14 |
| 最大回撤 | 6.38% |

## 风险提示

1. **持续超买超卖**：CCI可能长时间处于极端区域
2. **信号滞后**：指标基于历史数据存在滞后
3. **阈值调整**：不同市场可能需要不同的阈值
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数优化**：根据不同品种调整周期和阈值
2. **趋势过滤**：结合ADX或移动平均线判断趋势
3. **止损机制**：添加ATR止损保护
4. **仓位管理**：根据CCI强度动态调整仓位
5. **多重确认**：增加成交量或其他指标确认
