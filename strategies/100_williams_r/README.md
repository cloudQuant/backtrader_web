# 威廉指标策略 (Williams %R Strategy)

## 策略简介

威廉%R指标策略是由Larry Williams开发的动量指标策略，用于识别市场的超买和超卖状态，从而产生交易信号。

## 策略原理

### 指标构成

**威廉%R指标 (Williams %R)**
- 计算公式：-100 × (最高价 - 收盘价) / (最高价 - 最低价)
- 取值范围：0 到 -100
- 默认周期：14日

指标解读：
- -20以上：超买区域
- -80以下：超卖区域

### 交易逻辑

#### 入场条件（做多）
- Williams %R < -80（进入超卖区域）
- 指标开始上升（显示反转迹象）
- 避免接飞刀的策略

#### 出场条件
- Williams %R > -20（进入超买区域）

### 策略特点

- 仅做多（Long-only）
- 均值回归策略
- 等待反转确认后入场
- 类似于随机指标但缩放方式不同

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| period | 14 | Williams %R计算周期 |
| oversold | -80 | 超卖阈值 |
| overbought | -20 | 超买阈值 |

## 适用场景

### 最佳适用
- 震荡行情：价格在区间内波动
- 均值回归市场：价格倾向于回归均值
- 波动率适中的市场

### 不适用场景
- 强趋势行情：可能过早退出
- 持续创新高或新低的市场
- 跳空剧烈的市场

## 回测结果

使用Oracle (ORCL) 2010-2014年数据回测：

| 指标 | 数值 |
|------|------|
| 初始资金 | $100,000 |
| 最终价值 | $100,102.86 |
| 年化收益率 | 0.021% |
| 夏普比率 | 0.48 |
| 最大回撤 | 9.84% |

## 风险提示

1. **趋势行情风险**：在强趋势中表现不佳
2. **信号滞后**：指标基于历史数据存在滞后
3. **超卖持续**：超卖状态可能持续很久
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数调整**：根据市场波动率调整周期和阈值
2. **趋势过滤**：加入ADX或移动平均线判断趋势
3. **止损机制**：添加ATR止损保护
4. **仓位管理**：根据指标强度动态调整仓位
5. **多重确认**：结合成交量或其他指标确认
