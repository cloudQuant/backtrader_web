# 扩展交叉策略 (Extended Cross Strategy)

## 策略简介

扩展交叉策略是一个改进的移动平均线交叉策略，它引入ATR（平均真实波幅）来扩展交叉信号，减少假信号的产生。该策略还使用长期EMA作为趋势过滤器。

## 策略原理

### 核心思想
- 传统的MA交叉策略容易产生假信号
- 使用ATR扩展交叉阈值，只有在价格突破一定波动范围时才产生信号
- 长期EMA用于确认大趋势方向

### 交易规则
**买入条件：**
- 快速EMA（5周期）> 慢速EMA（20周期）+ ATR
- 收盘价 > 长期EMA（50周期）

**卖出条件：**
- 快速EMA（5周期）< 慢速EMA（20周期）- ATR
- 收盘价 < 长期EMA（50周期）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| ma1 | 5 | 快速EMA周期 |
| ma2 | 20 | 慢速EMA周期 |
| ma3 | 50 | 长期EMA周期（趋势过滤） |
| atr_mult | 1 | ATR扩展倍数 |

## 适用场景

- 趋势明显的市场
- 波动性较大的品种
- 需要减少假信号的交易场景

## 风险提示

1. **趋势反转风险**：在快速反转的市场中可能产生较大损失
2. **ATR参数敏感性**：ATR倍数需要根据市场波动性调整
3. **滞后性**：EMA指标的滞后性可能导致入场过晚
4. **震荡市损失**：在横盘震荡中可能频繁止损

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1208
- 夏普比率：-0.818
- 年化收益率：-0.02%
- 最大回撤：17.83%
- 最终资金：99,898.01
