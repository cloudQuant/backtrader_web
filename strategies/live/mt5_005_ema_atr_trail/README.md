# MT5 EMA+ATR移动止损策略

基于EMA趋势方向和ATR动态止损的策略，适用于MT5外汇品种。

## 策略逻辑

- **EMA趋势过滤**：价格在EMA上方时确认上升趋势，开多
- **ATR移动止损**：基于ATR动态计算止损距离，跟踪最高价
- **止损触发平仓**：价格跌破移动止损线时平仓

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ema_period | 50 | EMA趋势过滤周期 |
| atr_period | 14 | ATR计算周期 |
| atr_multiplier | 2.0 | ATR止损距离倍数 |
| volume | 0.01 | 交易手数 |

## 适用品种

AUDUSD、NZDUSD 等波动适中的外汇品种，推荐H1周期。
