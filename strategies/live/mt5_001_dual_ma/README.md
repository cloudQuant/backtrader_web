# MT5 双均线交叉策略

经典双均线交叉策略，适用于 MT5 外汇品种。

## 策略逻辑

- **金叉买入**：短期均线上穿长期均线时开多
- **死叉卖出**：短期均线下穿长期均线时平仓

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| short_period | 10 | 短期均线周期 |
| long_period | 30 | 长期均线周期 |
| volume | 0.01 | 交易手数 |

## 适用品种

EURUSD、GBPUSD、USDJPY 等主流外汇品种。

## 使用方式

1. 在 `config.yaml` 中填写 MT5 账号信息
2. 通过 backtrader_web 实盘管理页面启动
