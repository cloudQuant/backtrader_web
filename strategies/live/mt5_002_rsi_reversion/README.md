# MT5 RSI均值回归策略

基于RSI超买超卖的均值回归策略，适用于MT5外汇品种。

## 策略逻辑

- **超卖买入**：RSI低于超卖阈值时开多
- **超买卖出**：RSI高于超买阈值时平仓

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| rsi_period | 14 | RSI计算周期 |
| oversold | 30 | 超卖阈值 |
| overbought | 70 | 超买阈值 |
| volume | 0.01 | 交易手数 |

## 适用品种

GBPUSD、EURUSD 等震荡特性明显的外汇品种。
