# Alligator策略 (Alligator Strategy)

## 策略简介

Alligator（鳄鱼）指标是Bill Williams开发的技术分析工具，由三条平滑移动平均线组成，用于识别市场趋势和潜在的买卖时机。该指标形象地用鳄鱼的行为来比喻市场状态。

## 策略原理

### 核心思想
- 鳄鱼的下颚（蓝色线）、牙齿（红色线）、嘴唇（绿色线）分别代表不同周期的平滑移动平均
- 当三条线纠缠在一起时，鳄鱼在"休眠"（市场震荡）
- 当三条线分开时，鳄鱼在"进食"（趋势形成）
- 价格突破下颚线是趋势信号

### Alligator组成
- **下颚线（Jaw）**：13周期平滑移动平均线，代表市场的长期平衡
- **牙齿线（Teeth）**：8周期平滑移动平均线，代表市场的中期趋势
- **嘴唇线（Lips）**：5周期平滑移动平均线，代表市场的短期动量

### 交易规则
**买入条件：**
- 收盘价 > 鳄鱼下颚线

**卖出条件：**
- 收盘价 < 鳄鱼下颚线

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| jaw_period | 13 | 下颚线周期 |
| teeth_period | 8 | 牙齿线周期 |
| lips_period | 5 | 嘴唇线周期 |

## 适用场景

- 趋势明显的市场环境
- 适合中长线持仓
- 流动性好的主流品种

## 风险提示

1. **震荡市损失**：在横盘震荡中三条线频繁交叉，产生假信号
2. **趋势反转**：快速反转可能导致较大亏损
3. **滞后性**：平滑移动平均线存在滞后，可能错过最佳点位
4. **单一信号依赖**：仅依赖下颚线可能不够全面

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1245
- 夏普比率：0.047
- 年化收益率：0.002%
- 最大回撤：13.53%
- 最终资金：100,011.20
