# 分时均线策略 (TimeLine MA Strategy)

## 策略简介

分时均线策略是一个基于时间均价线和趋势过滤的日内交易策略。该策略计算当日累积平均价格作为时间均价线，结合移动平均线判断趋势方向，在价格突破时间均价线时进行交易。

## 策略原理

### 核心思想
1. **时间均价线**: 计算当日所有收盘价的累积平均值，代表市场平均成本
2. **趋势过滤**: 使用均线判断当前趋势方向
3. **突破信号**: 价格突破时间均价线作为入场信号
4. **追踪止损**: 使用移动止损保护利润

### 入场条件
- **做多**: MA向上 + 价格>MA + 价格从下向上突破时间均价线
- **做空**: MA向下 + 价格<MA + 价格从上向下跌破时间均价线

### 出场条件
- 价格反向突破时间均价线
- 追踪止损触发
- 14:55强制平仓

### 交易时间
- 允许开仓时间: 21:00-23:00, 9:00-11:00
- 强制平仓时间: 14:55

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ma_period | 200 | 均线周期 |
| stop_mult | 1 | 追踪止损百分比(%) |

## 适用场景

- 螺纹钢等期货品种
- 日内交易，避免隔夜风险
- 具有明显趋势的市场环境

## 风险提示

1. 本策略为日内策略，收盘前会强制平仓
2. 追踪止损可能在快速反转市场中失效
3. 参数需要根据不同品种调整
4. 实盘交易需考虑滑点、手续费等成本
