# 定时器双均线策略 (Timer Strategy)

## 策略简介

本策略实现了双均线交叉交易系统，同时演示了Backtrader的定时器(Timer)功能。策略在交易时段开始时触发定时器事件，用于执行周期性任务。

## 策略原理

1. **技术指标**：
   - 快速均线：10周期SMA
   - 慢速均线：30周期SMA
2. **买入信号**：快速均线向上穿越慢速均线
3. **卖出信号**：快速均线向下穿越慢速均线
4. **定时器功能**：在每次交易时段开始时触发事件

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| when | SESSION_START | 定时器触发时间 |
| timer | true | 是否启用定时器 |
| fast_period | 10 | 快速均线周期 |
| slow_period | 30 | 慢速均线周期 |

## 适用场景

- 趋势跟踪交易
- 需要定时执行任务的策略
- 演示定时器功能的使用

## 风险提示

- 双均线策略在震荡市场中会产生频繁交易
- 定时器事件可能影响策略性能
- 建议结合其他指标过滤信号
