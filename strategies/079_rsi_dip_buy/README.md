# RSI抄底策略 (RSI Dip Buy Strategy)

## 策略简介

RSI抄底策略基于RSI指标的中位突破来识别动量转换。当RSI从下方突破50时表明多头力量增强，策略选择买入；当RSI跌破45或触及止损/止盈时选择卖出。

## 策略原理

### 核心思想
- RSI指标以50为多空分界线
- RSI向上突破50表明多头力量占优
- 设置止损止盈控制风险

### 交易规则
**买入条件：**
- RSI[-1] <= 50 且 RSI[0] > 50（RSI向上突破50）

**卖出条件（满足任一）：**
- RSI < 45（动量转弱）
- 价格 < 买入价 * (1 - 0.5%)（止损）
- 价格 > 买入价 * (1 + 0.5%)（止盈）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| rsi_period | 10 | RSI计算周期 |
| rsi_buy | 50 | RSI买入阈值 |
| rsi_sell | 45 | RSI卖出阈值 |
| stop_loss | 0.005 | 止损幅度（0.5%） |
| take_profit | 0.005 | 止盈幅度（0.5%） |

## 适用场景

- 震荡市或轻微趋势市场
- 波动性适中的品种
- 短期交易

## 风险提示

1. **频繁止损**：0.5%的止损容易被触发
2. **RSI滞后**：RSI是滞后指标，可能错过最佳点位
3. **横盘损失**：在震荡中频繁开平仓可能产生损失
4. **止盈过小**：0.5%的止盈可能限制盈利空间

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1247
- 夏普比率：-0.633
- 年化收益率：-0.021%
- 最大回撤：16.15%
- 最终资金：99,893.93
