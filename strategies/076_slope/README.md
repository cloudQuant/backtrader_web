# 斜率策略 (Slope Strategy)

## 策略简介

斜率策略是一个简单直观的趋势跟踪策略，通过监测移动平均线的斜率方向来判断市场趋势。当MA上升时认为处于上升趋势并买入，当MA下降时认为处于下降趋势并卖出。

## 策略原理

### 核心思想
- 移动平均线的斜率反映了当前趋势方向
- 上升的MA表示上升趋势
- 下降的MA表示下降趋势

### 交易规则
**买入条件：**
- MA[0] > MA[-1]（当前MA值大于前一周期，斜率为正）

**卖出条件：**
- MA[0] < MA[-1]（当前MA值小于前一周期，斜率为负）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| ma_period | 14 | 移动平均线周期 |

## 适用场景

- 趋势明显的市场
- 中短期趋势跟踪
- 波动性适中的品种

## 风险提示

1. **频繁交易**：MA斜率频繁变化可能导致过度交易
2. **震荡损失**：在横盘震荡中容易产生连续小亏损
3. **滞后性**：MA是滞后指标，可能错过趋势转折点
4. **交易成本**：频繁交易会增加手续费成本

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1244
- 夏普比率：0.170
- 年化收益率：0.006%
- 最大回撤：10.38%
- 最终资金：100,028.49
