# Price Channel Strategy (价格通道突破策略)

## 策略简介

价格通道突破策略是一种经典的趋势跟踪策略，通过价格突破近期高点来识别上升趋势，通过跌破近期低点来识别趋势反转。该策略简单有效，是许多突破策略的基础。

## 策略原理

### 价格通道

价格通道由一定周期内的最高价和最低价构成：
- **通道上轨**：N日内的最高价
- **通道下轨**：M日内的最低价

### 入场条件
- 当前最高价 >= N日通道上轨（创新高）
- 开仓做多

### 出场条件
- 当前最低价 <= M日通道下轨（创新低）
- 平仓离场

## 策略参数

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| stake | 10 | 每次交易的股票数量 |
| entry_period | 20 | 入场通道周期（最高价） |
| exit_period | 10 | 出场通道周期（最低价） |

## 适用场景

- **趋势市场**：在明显的趋势行情中效果最佳
- **突破行情**：捕捉价格突破关键位置的行情
- **中长线交易**：适合中长期持仓

## 风险提示

1. **假突破**：价格可能短暂突破后迅速回落
2. **震荡亏损**：震荡市中容易产生连续亏损
3. **滑点风险**：突破时可能面临较大滑点
4. **参数敏感**：入场和出场周期需要仔细调整
