# 可转债周五高溢价轮动策略 (Convertible Bond Friday Rotation Strategy)

## 策略简介

可转债周五高溢价轮动策略是一个基于溢价率的定期轮动策略。该策略每周五根据溢价率选择可转债进行持仓，通过定期轮动实现收益。

## 策略原理

### 核心思想
1. **溢价率选股**: 选择溢价率最高的可转债（通常表示正股预期较强）
2. **定期轮动**: 每周五重新评估并调整持仓
3. **分散投资**: 同时持有3只可转债

### 交易逻辑
**每周五执行**:
1. 平掉所有现有持仓
2. 按溢价率排序所有可转债
3. 买入溢价率最高的3只可转债
4. 每只使用10%资金
5. 持有到下周五重新轮动

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| hold_num | 3 | 每周持有的可转债数量 |

## 适用场景

- 可转债市场
- 中长期投资
- 希望通过溢价率获得超额收益

## 风险提示

1. 高溢价率可能意味着高估值，存在回调风险
2. 每周轮动产生较多交易成本
3. 需要足够资金同时持有多只可转债
4. 溢价率策略在不同市场环境下效果差异较大
