# 菲阿里四价改进版策略 (Fei A'li Four-Price Strategy)

## 策略简介

菲阿里四价改进版策略是基于布林带过滤的日内突破策略。该策略使用前一日的高低点作为突破参考，结合布林带指标进行趋势过滤，只在符合趋势方向时进行突破交易。

## 策略原理

### 核心思想
1. **突破信号**: 价格突破前一日最高价或最低价
2. **趋势过滤**: 使用布林带中轨判断趋势方向
3. **突破确认**: 价格需突破布林带上下轨

### 入场条件
- **做多**: 价格 > 布林带上轨 AND 布林带中轨向上 AND 价格 > 前一日最高价
- **做空**: 布林带中轨向下 AND 价格 < 布林带下轨 AND 价格 < 前一日最低价

### 出场条件
- 在14:55平掉所有持仓（收盘前平仓）

### 交易时间
- 允许开仓时间: 21:00-23:00, 9:00-11:00
- 强制平仓时间: 14:55

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| boll_period | 200 | 布林带周期 |
| boll_mult | 2 | 布林带标准差倍数 |

## 适用场景

- 螺纹钢等波动较大的期货品种
- 日内交易，避免隔夜风险
- 趋势明显的市场环境

## 风险提示

1. 本策略为日内策略，收盘前会强制平仓
2. 突破策略在震荡市场中容易产生假突破
3. 需要充分测试参数，适应不同品种特性
4. 实盘交易需考虑滑点、手续费等交易成本
