# 双周期RSI策略 (Two Period RSI Strategy)

## 策略简介

这是Larry Connor开发的经典短期均值回归策略。与传统RSI策略不同，本策略使用极短的2周期RSI来捕捉超卖反弹机会。

## 策略原理

1. **趋势过滤**：价格必须位于200日均线之上
2. **买入条件**：2周期RSI < 5 (超卖)
3. **卖出条件**：收盘价突破5日均线

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| rsi_period | 2 | RSI周期 |
| sma_short | 5 | 卖出信号短期均线 |
| sma_long | 200 | 趋势过滤长期均线 |
| rsi_buy_threshold | 5 | RSI买入阈值 |

## 适用场景

- 上升趋势中的短期回调
- 波动率较高的股票
- 日内或短线交易

## 风险提示

- 仅适用于上升趋势
- 超卖后可能继续下跌
- 需要严格控制止损

## 策略优势

- 在上升趋势中表现良好
- 能够捕捉短期反弹
- 参数少，易于实现
