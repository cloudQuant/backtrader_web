# 随机指标策略 (Stochastic Oscillator Strategy)

## 策略简介

随机指标策略（Stochastic Oscillator）是由George Lane开发的经典技术分析工具，通过比较收盘价与一定时期内价格范围的关系，来识别超买和超卖状态。

## 策略原理

### 指标构成

**随机指标 (Stochastic Oscillator / KD指标)**
- %K线：(收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
- %D线：%K的M日平滑移动平均
- 默认参数：N=14，M=3

指标解读：
- 80以上：超买区域
- 20以下：超卖区域

### 交易逻辑

#### 入场条件（做多）
- K线向上穿越D线（金叉）
- K线在超卖区域（K < 20）
- 两个条件同时满足

#### 出场条件
- K线向下穿越D线（死叉）
- K线在超买区域（K > 80）
- 两个条件同时满足

### 策略特点

- 仅做多（Long-only）
- 结合交叉信号和区域判断
- 过滤中性区域的虚假信号
- 适合震荡行情

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| period | 14 | %K线计算周期 |
| period_dfast | 3 | %D线平滑周期 |
| oversold | 20 | 超卖阈值 |
| overbought | 80 | 超买阈值 |

## 适用场景

### 最佳适用
- 震荡行情：价格在区间内波动
- 均值回归市场
- 波动率适中的市场

### 不适用场景
- 强趋势行情：可能过早退出或错过趋势
- 持续创新高或新低的市场
- 跳空剧烈的市场

## 回测结果

使用Oracle (ORCL) 2010-2014年数据回测：

| 指标 | 数值 |
|------|------|
| 初始资金 | $100,000 |
| 最终价值 | $100,219.02 |
| 年化收益率 | 0.044% |
| 夏普比率 | 0.69 |
| 最大回撤 | 8.5% |

## 风险提示

1. **趋势行情风险**：在强趋势中表现不佳
2. **信号滞后**：指标基于历史数据存在滞后
3. **双重条件**：需要同时满足交叉和区域条件，信号较少
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数调整**：根据市场波动率调整周期和阈值
2. **趋势过滤**：加入ADX或移动平均线判断趋势
3. **宽松条件**：考虑去掉区域条件只保留交叉信号
4. **止损机制**：添加ATR止损保护
5. **仓位管理**：根据指标位置动态调整仓位
