# 三均线交叉策略 (Triple Cross Moving Average Crossover Strategy)

## 策略简介

三均线交叉策略是一种趋势跟踪策略，通过三条不同周期的均线的对齐情况来判断趋势方向，比双均线策略更加可靠。

## 策略原理

1. **均线设置**：
   - 短期均线：5周期SMA
   - 中期均线：8周期SMA
   - 长期均线：13周期SMA

2. **交易信号**：
   - **买入**：短期 > 中期 > 长期 (多头排列)
   - **卖出**：短期 < 中期 < 长期 (空头排列)

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| ma1_period | 5 | 短期均线周期 |
| ma2_period | 8 | 中期均线周期 |
| ma3_period | 13 | 长期均线周期 |

## 适用场景

- 明显的趋势行情
- 中长期趋势跟踪
- 股票、期货、外汇等市场

## 与双均线的区别

| 特点 | 双均线 | 三均线 |
|------|--------|--------|
| 信号数量 | 较多 | 较少但更可靠 |
| 假信号 | 较多 | 较少 |
| 反应速度 | 较快 | 较慢 |
| 趋势确认 | 一般 | 更强 |

## 风险提示

- 震荡市中表现不佳
- 信号较少，可能错过部分行情
- 均线存在滞后性

## 策略优势

- 三线排列确认趋势更加可靠
- 减少假突破交易
- 适合大趋势跟踪
