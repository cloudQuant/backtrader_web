# 唐奇安通道策略 (Donchian Channel Strategy)

## 策略简介

唐奇安通道是一种经典的趋势跟踪指标，由Richard Donchian开发。策略通过突破通道上下轨来捕捉趋势行情。

## 策略原理

1. **通道计算**：
   - 上轨：过去N天的最高价
   - 下轨：过去N天的最低价
   - 中轨：(上轨 + 下轨) / 2

2. **交易信号**：
   - **买入**：收盘价突破上轨
   - **卖出**：收盘价跌破下轨

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| period | 20 | 通道计算周期 |

## 适用场景

- 趋势明显的市场
- 突破交易策略
- 期货和外汇市场

## 风险提示

- 在震荡市中频繁假突破
- 突破后可能快速回调
- 需要配合止损策略

## 策略优势

- 简单易懂，参数少
- 能够捕捉大趋势行情
- 著名的"海龟交易法"基础
