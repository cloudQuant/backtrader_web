# Triple EMA Strategy (三重指数移动平均线策略)

## 策略简介

三重指数移动平均线策略是一种使用三条不同周期EMA来识别趋势方向的趋势跟踪策略。通过观察三条EMA的排列关系，可以更准确地判断当前趋势状态，避免单一指标的滞后性。

## 策略原理

### EMA排列关系

EMA（指数移动平均线）对近期价格赋予更高权重，反应更快：

- **多头排列**：EMA5 > EMA10 > EMA20
  - 短期均线在中期均线之上
  - 中期均线在长期均线之上
  - 表示强势上升趋势

- **空头排列**：EMA5 < EMA10 < EMA20
  - 短期均线在中期均线之下
  - 中期均线在长期均线之下
  - 表示强势下降趋势

### 入场条件
- EMA5 > EMA10 > EMA20（多头排列形成）
- 开仓做多

### 出场条件
- EMA5 < EMA10 < EMA20（空头排列形成）
- 平仓离场

## 策略参数

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| stake | 10 | 每次交易的股票数量 |
| fast | 5 | 快速EMA周期 |
| mid | 10 | 中速EMA周期 |
| slow | 20 | 慢速EMA周期 |

## 适用场景

- **趋势市场**：在明显的趋势行情中效果最佳
- **中短线交易**：适合中短期持仓
- **趋势确认**：三线排列可确认趋势强度

## 风险提示

1. **震荡亏损**：横盘震荡中容易产生虚假信号
2. **滞后性**：EMA仍有滞后，可能错过最佳点位
3. **参数敏感**：不同品种需要调整周期参数
4. **频繁交易**：震荡市可能产生多次小亏损
