# MACD梯度策略 (MACD Gradient Strategy)

## 策略简介

MACD梯度策略是一个基于MACD指标动量的趋势跟踪策略。该策略通过检测MACD指标的连续上升或连续下降来识别市场动量变化，从而产生交易信号。

## 策略原理

### 核心思想
- MACD的连续变化反映了市场动量的积累
- 连续三期上升表明上涨动量强劲
- 连续三期下降表明下跌动量强劲

### 交易规则
**买入条件：**
- MACD[0] > MACD[-1] > MACD[-2]（连续三期上升）

**卖出条件：**
- MACD[0] < MACD[-1] < MACD[-2]（连续三期下降）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| period_me1 | 12 | MACD快速EMA周期 |
| period_me2 | 26 | MACD慢速EMA周期 |

## 适用场景

- 趋势明显的市场环境
- 波动性适中的品种
- 适合捕捉中短期动量机会

## 风险提示

1. **动量衰竭风险**：动量可能突然衰竭，导致策略失败
2. **横盘损失**：在震荡市场中容易产生频繁交易和损失
3. **滞后性**：MACD是滞后指标，可能错过趋势的初始阶段
4. **假信号**：短期波动可能产生连续变化的假信号

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1224
- 夏普比率：-0.139
- 年化收益率：-0.005%
- 最大回撤：13.84%
- 最终资金：99,975.90
