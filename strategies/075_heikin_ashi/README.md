# Heikin Ashi策略 (Heikin Ashi Strategy)

## 策略简介

Heikin Ashi策略结合了Heikin Ashi蜡烛图和简单移动平均线交叉信号，用于趋势跟踪交易。Heikin Ashi蜡烛图能够更好地过滤市场噪音，识别趋势方向。

## 策略原理

### 核心思想
- Heikin Ashi蜡烛图通过特殊的计算方式平滑价格波动
- 结合MA交叉系统确认趋势变化
- 金叉买入，死叉平仓

### 交易规则
**买入条件：**
- 短期SMA（10周期）上穿长期SMA（30周期）（金叉）

**平仓条件：**
- 短期SMA（10周期）下穿长期SMA（30周期）（死叉）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |

## 适用场景

- 趋势明确的市场环境
- 适合中长期持仓
- 适用于波动较大的品种

## 风险提示

1. **趋势识别延迟**：MA交叉信号存在滞后，可能错过最佳入场点
2. **震荡市损失**：在横盘震荡中容易产生频繁交易
3. **假突破风险**：短期突破可能是假信号
4. **单向交易**：策略只做多，不做空

- 卖出次数：22
- 总交易次数：23
- 夏普比率：0.575
- 年化收益率：0.018%
- 最大回撤：8.76%
- 最终资金：100,091.06
