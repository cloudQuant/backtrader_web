# 双指数移动平均交叉策略 (DEMA Crossover Strategy)

## 策略简介

双指数移动平均交叉策略（DEMA Crossover）是一种趋势跟踪策略，利用两条不同周期的双指数移动平均线（DEMA）交叉产生交易信号。

DEMA由Patrick Mulloy开发，旨在减少传统移动平均线的滞后性，通过更复杂的计算公式实现两次平滑处理。

## 策略原理

### 指标构成

**双指数移动平均线（DEMA）**
- 快速DEMA：默认5日周期
- 慢速DEMA：默认21日周期
- 交叉指标：追踪快速DEMA与慢速DEMA的交叉关系

### 交易逻辑

#### 入场条件（做多）
- 快速DEMA向上穿越慢速DEMA（金叉）
- crossover > 0

#### 出场条件
- 快速DEMA向下穿越慢速DEMA（死叉）
- crossover < 0

### 策略特点

- 仅做多（Long-only）
- 趋势跟踪策略
- 简单清晰的买卖信号
- 使用平仓订单离场

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| fast_period | 5 | 快速DEMA周期 |
| slow_period | 21 | 慢速DEMA周期 |

## 适用场景

### 最佳适用
- 明显趋势行情：能够捕捉趋势的开始和结束
- 波动适中的市场：避免频繁交叉产生虚假信号

### 不适用场景
- 震荡行情：频繁交叉导致连续亏损
- 高波动市场：信号滞后性可能放大损失

## 风险提示

1. **震荡市场亏损**：在横盘震荡中频繁交易导致损失
2. **信号滞后**：趋势行情启动和结束时的信号都存在滞后
3. **无止损机制**：策略未设置止损，最大亏损可能较大
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数优化**：调整快慢周期以适应不同市场
2. **趋势过滤**：加入ADX等指标判断趋势强度
3. **止损机制**：添加止损或ATR-based止损
4. **仓位管理**：根据信号强度动态调整仓位
5. **多时间框架**：结合更高时间框架趋势确认
