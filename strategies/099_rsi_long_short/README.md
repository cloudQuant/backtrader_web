# RSI长短周期策略 (RSI Long/Short Strategy)

## 策略简介

RSI长短周期策略使用两个不同周期的RSI指标组合来判断交易时机。长周期RSI用于确认整体趋势强度，短周期RSI用于精准的入场和出场时机把握。

## 策略原理

### 指标构成

1. **长周期RSI**
   - 默认周期：14日
   - 作用：确认整体趋势方向

2. **短周期RSI**
   - 默认周期：5日
   - 作用：提供入场和出场的精准时机

### 交易逻辑

#### 入场条件（做多）
- 长周期RSI > 50（趋势向上）
- 短周期RSI > 65（短期强势）
- 两个条件同时满足

#### 出场条件
- 短周期RSI < 45（短期动能减弱）

### 策略特点

- 仅做多（Long-only）
- 双时间框架确认
- 过滤虚假信号
- 灵活的出场机制

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位 |
| period_long | 14 | 长周期RSI计算周期 |
| period_short | 5 | 短周期RSI计算周期 |
| buy_rsi_long | 50 | 长周期RSI买入阈值 |
| buy_rsi_short | 65 | 短周期RSI买入阈值 |
| sell_rsi_short | 45 | 短周期RSI卖出阈值 |

## 适用场景

### 最佳适用
- 趋势明确的市场
- 需要精准入场时机的交易
- 波动率适中的行情

### 不适用场景
- 持续横盘震荡
- 极端波动市场
- 快速反转行情

## 风险提示

1. **双重滞后**：两个RSI指标都存在滞后性
2. **错过趋势**：保守的入场条件可能错过部分机会
3. **过早离场**：仅基于短周期RSI出场可能错过后续趋势
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数调整**：根据市场特性调整长短周期比例
2. **止损机制**：添加ATR或固定止损
3. **仓位管理**：根据RSI强度动态调整仓位
4. **趋势确认**：加入ADX或MACD辅助判断
5. **出场优化**：结合价格位置或移动平均线作为出场依据
