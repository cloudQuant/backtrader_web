# SuperTrend策略 (SuperTrend Strategy)

## 策略简介

SuperTrend策略是一个基于平均真实波幅（ATR）的趋势跟踪策略。SuperTrend指标通过ATR计算动态支撑阻力位，能够根据市场波动性调整跟踪止损距离，是流行的趋势跟踪工具。

## 策略原理

### 核心思想
- 使用ATR计算动态支撑阻力位
- 根据价格与SuperTrend线的相对位置确定趋势方向
- 趋势转换时产生交易信号

### SuperTrend计算方法
1. 计算HL2 = (最高价 + 最低价) / 2
2. 计算上轨 = HL2 + (ATR × 倍数)
3. 计算下轨 = HL2 - (ATR × 倍数)
4. 根据前一日趋势和价格关系确定当前SuperTrend值

### 交易规则
**买入条件：**
- SuperTrend方向由-1（下跌）转为1（上涨）

**卖出条件：**
- SuperTrend方向由1（上涨）转为-1（下跌）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| period | 10 | ATR计算周期 |
| multiplier | 3.0 | ATR倍数（带宽系数） |

## 适用场景

- 趋势明显的市场环境
- 波动性较大的品种
- 中长期趋势跟踪

## 风险提示

1. **震荡损失**：在横盘震荡中容易产生频繁止损
2. **趋势反转**：快速反转可能导致较大亏损
3. **参数敏感**：ATR周期和倍数需要根据品种特性调整
4. **入场延迟**：趋势确认后入场可能错过部分利润

## 回测结果

基于Oracle（ORCL）股票数据（2010-2014）回测：
- 总交易天数：1247
- 夏普比率：-0.004
- 年化收益率：-0.0002%
- 最大回撤：11.22%
- 最终资金：99,999.23
