# 简单RSI策略 (Simple RSI Strategy)

## 策略简介

简单RSI策略是一个结合了RSI（相对强弱指标）超买超卖信号与EMA（指数移动平均线）趋势过滤的交易策略。该策略在价格下跌到超卖区域且趋势向上时买入，在价格上涨到超买区域时卖出。

## 策略原理

### 核心思想
- RSI指标用于识别超买超卖状态
- EMA交叉用于确认趋势方向
- 只在趋势向上且RSI超卖时买入，避免逆势交易

### 交易规则
**买入条件：**
- RSI < 30（超卖）
- 快速EMA（10周期）> 慢速EMA（100周期）（上升趋势）

**卖出条件：**
- RSI > 70（超买）

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的数量 |
| period_ema_fast | 10 | 快速EMA周期 |
| period_ema_slow | 100 | 慢速EMA周期 |
| rsi_period | 14 | RSI计算周期 |
| rsi_oversold | 30 | RSI超卖阈值 |
| rsi_overbought | 70 | RSI超买阈值 |

## 适用场景

- 震荡向上的市场环境
- 具有明显趋势的股票或期货品种
- 日内或日线级别的交易

## 风险提示

1. **趋势反转风险**：在趋势快速反转时可能产生较大损失
2. **横盘震荡风险**：在震荡市中频繁交易可能增加交易成本
3. **参数敏感性**：RSI和EMA参数需要根据不同品种进行调整
4. **滞后性**：EMA指标存在滞后，可能在趋势转变时反应不及时
