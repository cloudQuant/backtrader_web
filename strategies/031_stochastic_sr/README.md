# 随机指标支撑阻力策略 (Stochastic Support/Resistance Strategy)

## 策略简介

随机指标支撑阻力策略利用随机指标识别超买超卖区域作为入场点，使用唐奇安通道确定历史支撑阻力位，并设置双重止损保护。

## 策略原理

### 指标说明

- **随机指标 (Stochastic Oscillator)**: 识别超买超卖状态
- **唐奇安通道 (Donchian Channel)**: 确定N日内的最高最低价作为支撑阻力

### 交易规则

**开仓条件：**
- 做空：随机指标D线从>=80区间跌破80（超买反转）
- 做多：随机指标D线从<=20区间突破20（超卖反转）

**平仓条件：**
- 多头平仓：随机指标D线 >= 70
- 空头平仓：随机指标D线 <= 30

**止损规则（双重止损）：**
- 固定点数止损：入场价 +/- stop_pips
- 唐奇安通道止损：N日最高/最低价

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period | 14 | 随机指标周期（也是唐奇安通道周期） |
| pfast | 3 | 快速均线周期 |
| pslow | 3 | 慢速均线周期 |
| upperLimit | 80 | 超买阈值 |
| lowerLimit | 20 | 超卖阈值 |
| stop_pips | 0.5 | 固定止损点数 |

## 适用场景

- 震荡市场或区间震荡
- 价格在明确支撑阻力位间波动
- 适合短线波段交易

## 风险提示

1. 强趋势突破时止损可能被触发
2. 超买超卖可能持续较长时间
3. 需要根据品种调整止损幅度

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：309笔
- 胜率：36.25%
- 最终资金：99,989.23元（初始100,000元）
- 年化收益率：接近0%
- 最大回撤：2.18%
