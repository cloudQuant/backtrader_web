# 布林带ADX均值回归策略 (BB + ADX Mean Reversion Strategy)

## 策略简介

布林带ADX均值回归策略使用布林带提供均值回归信号，同时使用ADX指标识别和避免趋势市场。仅在ADX低于阈值（非趋势市场）时执行均值回归交易。

## 策略原理

### 指标说明

- **布林带 (Bollinger Bands)**: 识别超买超卖状态
- **ADX (Average Directional Movement Index)**: 衡量趋势强度，过滤趋势市场

### 交易规则

**开仓条件（ADX < ADX_Max时）：**
- 做空：价格从上方跌破上轨（超买回归）
- 做多：价格从下方突破下轨（超卖回归）

**平仓条件：**
- 多头平仓：价格从下方突破中轨
- 空头平仓：价格从上方跌破中轨

**止损规则：**
- 做多止损：下轨价格
- 做空止损：上轨价格

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| BB_MA | 20 | 布林带均线周期 |
| BB_SD | 2 | 布林带标准差倍数 |
| ADX_Period | 14 | ADX周期 |
| ADX_Max | 40 | ADX阈值（超过则避免交易） |

## 适用场景

- 震荡市场或无明确趋势的市场
- 价格在合理区间内波动
- 适合区间交易策略

## 风险提示

1. 在趋势市场中会完全避免交易
2. 假突破可能造成损失
3. 参数需要根据市场特性调整

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：293笔
- 胜率：20.14%
- 最终资金：99,971.15元（初始100,000元）
- 年化收益率：接近0%
- 最大回撤：3.77%
