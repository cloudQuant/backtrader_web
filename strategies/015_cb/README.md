# 可转债多因子日内策略 (Convertible Bond Intraday Strategy)

## 策略简介

可转债多因子日内策略是一个基于多个技术因子筛选的可转债交易策略。该策略同时监控多个可转债品种，根据预设的因子条件筛选出符合条件的品种进行交易。

## 策略原理

### 核心思想
1. **多因子筛选**: 使用多个技术因子筛选交易标的
2. **分散投资**: 同时持有多个可转债降低风险
3. **严格止损**: 设置多种止盈止损条件

### 入场条件（需同时满足）
- 价格 > 20周期均线
- 价格涨跌幅在-1%到+1%之间
- 成交量 < 30周期均量的4倍
- 均线上升但增速放缓

### 出场条件（满足任一即触发）
- 持仓超过10个交易日
- 价格跌破20周期最低点
- 收益超过3%（止盈）
- 可转债到期

### 仓位管理
- 最多同时持有2只可转债
- 单只可转债使用40%资金

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ma_period | 20 | 均线周期 |
| can_trade_num | 2 | 最大可同时持有的可转债数量 |

## 适用场景

- 可转债市场
- 日内或短线交易
- 兼顾收益与风险的投资需求

## 风险提示

1. 可转债市场流动性相对较低
2. 多因子策略需要充分回测和参数优化
3. 同时持仓多个品种需要更大资金量
4. 实盘交易需考虑滑点、手续费等成本
