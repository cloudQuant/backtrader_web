# EMA双均线交叉策略 (EMA Dual Moving Average Crossover Strategy)

## 策略简介

EMA双均线交叉策略是一个经典的趋势跟踪策略。该策略使用两个不同周期的指数移动平均线（EMA），当快线上穿慢线时产生买入信号（金叉），当快线下穿慢线时产生卖出信号（死叉）。

## 策略原理

### 核心思想
1. **趋势识别**: 使用双EMA交叉识别趋势变化
2. **金叉做多**: 快线上穿慢线 → 开多仓
3. **死叉做空**: 快线下穿慢线 → 开空仓
4. **多周期过滤**: 使用日线数据过滤交易时机

### 交易逻辑
- **开多仓**: 死叉信号总和 = 1
- **开空仓**: 金叉信号总和 = -1
- **平多仓**: 死叉信号
- **平空仓**: 金叉信号

### 交叉信号计算
```
CrossOver = 1  (金叉: 快线上穿慢线)
CrossOver = -1 (死叉: 快线下穿慢线)
CrossOver = 0  (无交叉)
```

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| fast_period | 80 | 快线EMA周期 |
| slow_period | 200 | 慢线EMA周期 |
| short_size | 2 | 空头仓位规模 |
| long_size | 1 | 多头仓位规模 |

## 适用场景

- 股票、期货等金融品种
- 趋势明显的市场环境
- 需要多周期过滤的交易系统

## 风险提示

1. 双均线策略在震荡市场容易产生假信号
2. 滞后性：EMA是滞后指标，可能错过最佳入场点
3. 需要合理设置仓位规模以控制风险
4. 多周期数据需要确保同步正确
