# 布林带反转策略 (Bollinger Band Reversal Strategy)

## 策略简介

布林带反转策略是一种均值回归策略，与趋势跟踪策略相反。当价格超买（突破上轨）时做空，超卖（跌破下轨）时做多，预期价格会回归均值。

## 策略原理

### 指标说明

- **布林带 (Bollinger Bands)**: 用于识别超买超卖状态

### 交易规则

**开仓条件：**
- 做空：收盘价连续两根K线高于上轨（超买反转）
- 做多：收盘价连续两根K线低于下轨（超卖反转）

**平仓条件：**
- 多头平仓：收盘价从下方突破上轨
- 空头平仓：收盘价从上方跌破下轨

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period_boll | 52 | 布林带周期（较短的周期用于捕捉反转机会） |

## 适用场景

- 震荡市场或区间震荡行情
- 价格偏离均值后的回归行情
- 不适合强趋势市场

## 风险提示

1. 在强趋势市场中可能持续亏损
2. 反转信号可能只是趋势中的短暂回调
3. 需要配合趋势过滤指标使用

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：48笔
- 胜率：59.52%
- 最终资金：19,875.22元（初始100,000元）
- 年化收益率：-7.24%（亏损）
- 最大回撤：397.64%

**注意**: 该策略在测试数据上表现不佳，说明均值回归策略不适用于所有市场环境。
