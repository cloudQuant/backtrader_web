# ETF轮动策略 (ETF Rotation Strategy)

## 策略简介

ETF轮动策略是一个基于均线比率的资产配置策略。该策略在上证50ETF和创业板ETF之间进行动态轮动，根据动量强弱选择持仓标的。

## 策略原理

### 核心思想
1. **动量指标**: 使用价格/均线比率衡量动量强弱
2. **轮动选择**: 持有动量较强的ETF
3. **风险控制**: 当两个ETF都低于均线时清仓

### 交易逻辑
- **清仓条件**: 两个ETF都低于均线
- **选择上证50**: 上证50动量 > 创业板动量
- **选择创业板**: 创业板动量 > 上证50动量

### 动量计算
```
动量 = 当前价格 / MA价格
```

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ma_period | 20 | 均线周期 |

## 适用场景

- A股市场ETF投资
- 大小盘风格轮动
- 中长期趋势跟踪

## 风险提示

1. 轮动策略在震荡市场中可能频繁交易
2. 两个ETF同时下跌时策略会空仓
3. 需要考虑ETF交易成本和流动性
4. 历史表现不代表未来收益
