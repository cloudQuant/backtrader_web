# 布林带RSI策略 (Bollinger Bands + RSI Strategy)

## 策略简介

布林带RSI策略是一种经典的均值回归策略，结合了两个互补的技术指标：
- **布林带 (Bollinger Bands)**: 识别价格相对于近期波动率的极端位置
- **相对强弱指标 (RSI)**: 识别动量极端状态（超买/超卖条件）

## 策略原理

### 指标构成

1. **布林带**
   - 中轨：20日简单移动平均线 (SMA)
   - 上轨：中轨 + 2倍标准差
   - 下轨：中轨 - 2倍标准差

2. **RSI指标**
   - 14日相对强弱指标
   - 衡量价格变动的速度和幅度

### 交易逻辑

#### 入场条件（做多）
- RSI < 30（超卖状态）
- 收盘价 < 布林带下轨
- 两个条件同时满足（AND逻辑）

#### 出场条件
- RSI > 70（超买状态）OR 收盘价 > 布林带上轨
- 任一条件触发即平仓（OR逻辑）

### 策略特点

- 仅做多（Long-only）
- 固定仓位大小
- 同一时间只持有一个仓位
- 不加仓也不部分平仓

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易的固定仓位（股/合约数） |
| bb_period | 20 | 布林带周期 |
| bb_devfactor | 2.0 | 标准差倍数（2.0 = 2个标准差） |
| rsi_period | 14 | RSI计算周期 |
| rsi_oversold | 30 | RSI超卖阈值（买入信号） |
| rsi_overbought | 70 | RSI超买阈值（卖出信号） |

## 适用场景

### 最佳适用
- 震荡行情：价格在区间内波动
- 均值回归市场：价格倾向于回归均值
- 波动率适中的市场

### 不适用场景
- 强趋势行情：可能错过大趋势或过早离场
- 极端波动市场：布林带和RSI可能失真
- 跳空剧烈的市场：可能导致意外损失

## 风险提示

1. **趋势市场风险**：在强趋势行情中表现不佳
2. **过早离场**：使用OR逻辑可能导致过早退出
3. **信号滞后**：基于指标的交易存在天然滞后性
4. **固定仓位**：未考虑风险管理和资金管理

## 优化建议

1. **参数优化**：根据不同品种调整周期和阈值
2. **加入趋势过滤**：使用ADX等指标判断趋势强度
3. **仓位管理**：根据波动率动态调整仓位
4. **止损机制**：添加止损以控制下行风险
5. **多重确认**：增加成交量或其他指标确认信号
