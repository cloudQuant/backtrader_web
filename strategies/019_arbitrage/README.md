# 国债期货跨期套利策略 (Treasury Futures Spread Arbitrage Strategy)

## 策略简介

国债期货跨期套利策略是一个基于价差的统计套利策略。该策略通过同时交易近月和远月合约，在价差偏离正常区间时建立套利头寸，等待价差回归获利。

## 策略原理

### 核心思想
1. **价差均值回归**: 近月和远月价差会在合理范围内波动
2. **套利机会**: 当价差超出合理范围时产生套利机会
3. **自动换月**: 支持合约到期时自动换月

### 交易逻辑
- **开多价差**: 近月-远月价差 < 下限 → 买近月、卖远月
- **开空价差**: 近月-远月价差 > 上限 → 卖近月、买远月
- **平仓**: 价差回归到合理范围

### 价差计算
```
价差 = 近月合约价格 - 远月合约价格
```

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| spread_low | 0.06 | 价差下限（低于此值开多价差） |
| spread_high | 0.52 | 价差上限（高于此值开空价差） |

## 适用场景

- 国债期货等金融期货
- 具有明显价差规律的品种
- 市场中性套利需求

## 风险提示

1. 价差可能持续扩大导致浮亏扩大
2. 套利策略双边持仓需要更多保证金
3. 换月时可能产生额外成本
4. 极端市场环境下价差可能失效
