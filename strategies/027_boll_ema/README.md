# 布林带EMA组合策略 (Bollinger Bands + EMA Strategy)

## 策略简介

布林带EMA组合策略结合了布林带的突破信号和EMA的趋势确认功能。布林带提供入场时机，EMA确认趋势方向，布林带宽度用于过滤波动性。

## 策略原理

### 指标说明

- **布林带 (Bollinger Bands)**: 提供价格突破信号
- **指数移动平均线 (EMA)**: 确认趋势方向

### 交易规则

**开仓条件：**

做多（需同时满足）：
- 当前和前一根收盘价连续突破上轨
- EMA位于中轨上方
- 过去3根K线收盘价均高于中轨
- 布林带宽度 > boll_diff（波动率过滤）

做空（需同时满足）：
- 当前和前一根收盘价连续跌破下轨
- EMA位于中轨下方
- 过去3根K线收盘价均低于中轨
- 布林带宽度 > boll_diff（波动率过滤）

**平仓条件：**
- 多头平仓：触发止损 或 EMA跌破中轨
- 空头平仓：触发止损 或 EMA突破中轨

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period_boll | 136 | 布林带周期 |
| period_ema | 99 | EMA周期 |
| boll_diff | 0.5 | 最小布林带宽度（波动率过滤） |
| price_diff | 0.3 | 止损价格差 |

## 适用场景

- 趋势明确且波动率适中的市场
- 需要多重确认的稳健型投资者
- 适合中长期趋势跟踪

## 风险提示

1. 多重确认条件可能错过部分机会
2. 参数设置需要针对不同品种优化
3. 低波动环境下可能长期无交易

## 回测表现

基于上海股票(sh600000) 2000-2022年数据回测：
- 总交易次数：37笔
- 胜率：27.03%
- 最终资金：705,655.57元（初始100,000元）
- 年化收益率：9.52%
- 最大回撤：45.38%
