# Hans123日内突破策略 (Hans123 Intraday Breakout Strategy)

## 策略简介

Hans123策略是一个经典的日内突破策略，由知名交易员Hans开发。该策略通过开盘后一段时间内形成的价格区间进行突破交易，配合均线指标过滤趋势方向。

## 策略原理

### 核心思想
1. **区间确定**: 开盘后N根K线的高点作为上轨，低点作为下轨
2. **趋势过滤**: 使用均线判断趋势方向
3. **突破交易**: 价格突破区间且符合趋势方向时入场

### 入场条件
- **做多**: MA向上 + 价格>MA + 价格突破上轨
- **做空**: MA向下 + 价格<MA + 价格突破下轨

### 出场条件
- 在14:55平掉所有持仓（收盘前平仓）

### 交易时间
- 允许开仓时间: 21:00-23:00, 9:00-11:00
- 强制平仓时间: 14:55

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ma_period | 200 | 均线周期（用于趋势过滤） |
| bar_num | 2 | 开盘后用于计算高低点的K线数量 |

## 适用场景

- 螺纹钢等期货品种
- 日内交易，避免隔夜风险
- 具有明显趋势方向的市场

## 风险提示

1. 本策略为日内策略，收盘前会强制平仓
2. 震荡市场中容易产生假突破
3. 参数需要根据不同品种和市场环境调整
4. 实盘交易需考虑滑点、手续费等成本
