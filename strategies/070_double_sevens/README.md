# 双七策略 (Double Sevens Strategy)

## 策略简介

双七策略是Larry Connor开发的一种短期均值回归策略，通过在上升趋势中买入N日新低、卖出N日新高来获取收益。

## 策略原理

1. **趋势过滤**：价格位于200日或70日均线之上
2. **买入条件**：价格创N日新低(抄底)
3. **卖出条件**：价格创N日新高(止盈)

默认N=7，因此称为"双七"策略。

## 策略参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| stake | 10 | 每次交易数量 |
| period | 7 | N日新高/低周期 |
| sma_short | 70 | 短期趋势均线 |
| sma_long | 200 | 长期趋势均线 |

## 适用场景

- 上升趋势中的回调买入
- 波动率较高的个股
- 短期波段交易

## 风险提示

- 仅适用于上升趋势
- 下跌趋势中买入容易被套
- 新低后可能继续下跌
- 需要严格的止损策略

## 策略优势

- 在上升趋势中表现优秀
- 买入时点风险收益比高
- 参数简单，易于实现

## 与2周期RSI策略的区别

- 双七策略使用价格新低新高
- 2周期RSI使用RSI指标
- 双七策略持仓时间可能更短
