# 迭代127 - 策略交易工作区 - 总体架构设计

> **文档状态**: 优化版  
> **最后更新**: 2026-04-13

## 1. 设计目标

将现有"模拟交易"和"实盘交易"两个独立功能合并为统一的**策略交易工作区**，在 **最大化复用策略研究（迭代124）架构** 的前提下，新增实时交易管理能力。

核心改动：

1. **工作区类型扩展** — 在现有 Workspace 模型上增加 `workspace_type` 字段，区分 `research`（策略研究）和 `trading`（策略交易）
2. **策略单元交易扩展** — 在现有 StrategyUnit 上增加交易模式（`live`/`simulate`）、网关配置、运行时交易状态字段
3. **实时交易状态层** — 新增内存级交易运行时状态管理，支持头寸、盈亏、最新价等实时指标
4. **自动交易调度迁入** — 将现有 LiveTradingPage 的自动交易调度迁入工作区级别，但进入 P1
5. **侧边栏渐进替换** — 新增"策略交易"入口，旧"模拟交易"和"实盘交易"先保留用于灰度回退

## 4. 核心概念映射

### 4.1 策略交易 vs 策略研究对比

| 维度 | 策略研究 (research) | 策略交易 (trading) |
|------|---------------------|-------------------|
| 工作区类型 | `workspace_type = 'research'` | `workspace_type = 'trading'` |
| 运行引擎 | BacktestService | LiveTradingService / PaperTradingService |
| 运行模式 | 回测（历史数据） | 实时交易（实盘/模拟） |
| 结果类型 | 回测结果快照 | 实时运行时状态 |
| 状态更新 | 运行完成后更新 | 持续实时更新 |
| 额外配置 | 优化参数 | 网关/交易所配置、自动交易调度 |
| 特有功能 | 参数优化、组合报告 | 锁定交易、头寸管理、自动交易调度 |
| 共有功能 | CRUD、导入导出、数据源、单元设置、策略参数、排序、重命名 | 同左 |

### 4.2 策略单元交易模式

| 交易模式 | 说明 | 后端服务 | 网关需求 |
|----------|------|---------|---------|
| `simulate` | 模拟交易 | PaperTradingService | 不需要真实网关 |
| `live` | 实盘交易 | LiveTradingService | 需要网关预设 (GatewayPreset) |

## 5. 技术栈

- **前端**: Vue 3 + TypeScript + Element Plus + TailwindCSS + ECharts + Pinia
- **后端**: FastAPI + SQLAlchemy 2.0 + Pydantic v2
- **实盘引擎**: Backtrader + CCXT (复用现有 `LiveTradingService`)
- **模拟引擎**: Backtrader PaperTrading (复用现有 `PaperTradingService`)
- **实时通信**: REST API 轮询 (基础) + WebSocket (增强)
- **数据库**: SQLite/PostgreSQL/MySQL (与现有一致)

## 6. 设计原则

1. **最大化复用** — 策略交易与策略研究共享 Workspace/StrategyUnit 模型、前端组件、API层，通过 `workspace_type` 区分行为
2. **向下兼容** — Workspace 表扩展字段有默认值，不影响现有策略研究功能
3. **运行时与持久化分离** — 交易实时状态（头寸/盈亏/价格）保存在内存/Redis，不频繁写库；持久化数据（配置/历史汇总）入库
4. **渐进式替换** — 新功能开发完成并验证后再删除旧模拟/实盘页面，不一刀切
5. **先闭环再丰富** — P0 交付统一入口、手动启停、状态监控和研究互通，P1 补充自动交易/锁定/头寸管理，P2 顺延高级功能与旧入口彻底下线

## 7. 本迭代不做项（明确排除）

- 重写现有 LiveTradingService / PaperTradingService 执行引擎
- 实现复杂的多账户资金管理
- 实现风控引擎（止损止盈规则引擎）
- 实现自动化交易信号生成系统
- 右键菜单、列拖拽等桌面化交互
- 多交易所同时连接的高级编排
- 历史交易回放/对账功能
