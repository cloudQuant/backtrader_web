## 3. 工作包定义

### WP1：国际化翻译应用（P1）

#### 目标
将i18n翻译应用到核心Vue组件，实现中英文可切换。

#### 当前问题
- LoginPage.vue：硬编码"用户名"、"密码"、"登录"等
- DashboardPage.vue：硬编码"回测次数"、"策略数量"等
- 其他页面同样存在硬编码中文

#### 执行步骤

**1.1 优先翻译页面**
1. LoginPage.vue - 登录页面
2. RegisterPage.vue - 注册页面
3. DashboardPage.vue - 仪表盘
4. BacktestPage.vue - 回测页面
5. StrategyPage.vue - 策略页面

**1.2 翻译模式**
```vue
<template>
  <!-- 替换硬编码 -->
  <span>{{ $t('auth.login') }}</span>
  <!-- 替换placeholder -->
  <el-input :placeholder="$t('auth.username')" />
  <!-- 替换表单验证消息 -->
  { required: true, message: t('auth.invalidCredentials'), trigger: 'blur' }
</template>

<script setup lang="ts">
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>
```

#### 验收标准
- [ ] 核心页面使用$t()翻译
- [ ] 中英文切换生效
- [ ] 表单验证消息国际化

### WP2：响应式布局应用（P1）

#### 目标
将响应式mixin应用到核心页面，优化移动端体验。

#### 当前问题
- _responsive.scss已定义但未被使用
- 页面在移动端显示挤压

#### 执行步骤

**2.1 导入响应式样式**
```scss
// 在style.css或组件中导入
@use './styles/responsive' as *;
```

**2.2 优化核心页面**
- DashboardPage：移动端堆叠卡片
- BacktestPage：表单单列布局
- LiveTradingPage：表格卡片化

**2.3 添加移动端样式**
```vue
<style scoped lang="scss">
@use '@/styles/responsive' as *;

.stat-card {
  @include respond-to('sm') {
    margin-bottom: 12px;
  }
}
</style>
```

#### 验收标准
- [ ] 核心页面在375px宽度可用
- [ ] 无横向滚动问题
