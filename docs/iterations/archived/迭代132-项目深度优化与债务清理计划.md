### WP1：静默异常清理（P0）

#### 目标
替换所有 `except Exception: pass` 为有意义的异常处理。

#### 执行原则
1. **关键路径**：必须记录日志并考虑是否需要抛出
2. **清理操作**：可保留静默，但添加注释说明原因
3. **数据解析**：记录警告日志，使用默认值

#### 修改清单

**高优先级（核心业务逻辑）**：

```python
# services/live_execution_service.py:179
# 修改前
except Exception:
    pass
# 修改后
except Exception as e:
    logger.warning("Error waiting for process: %s", e)

# services/live_execution_service.py:203
# 修改前
except Exception:
    pass
# 修改后
except Exception as e:
    logger.warning("Failed to decode stderr: %s", e)
```

**中优先级（辅助功能）**：

```python
# services/backtest_service.py:302
# 修改前
except Exception:
    pass
# 修改后
except Exception as e:
    logger.debug("Cleanup dir failed (ignored): %s", e)
```
