# 快速上手

5分钟内完成您的首次回测！

## 前置条件

- [安装指南](./installation.md) 已完成
- 服务已启动（后端 + 前端）

## 步骤 1：访问平台

打开浏览器访问：
- **前端**：http://localhost:8080 (开发)
- **后端 API 文档**：http://localhost:8000/docs

## 步骤 2：注册账户

1. 点击前端页面的"注册"
2. 输入用户名、邮箱和密码
3. 使用新账户登录

或直接使用 API：

```bash
curl -X POST http://localhost:8000/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"testuser","email":"test@example.com","password":"testpass123"}'
```

## 步骤 3：选择策略

从 **118 个内置策略模板** 中选择，包括：

### 趋势跟踪
- 双均线
- MACD
- 海龟交易法

### 均值回归
- 布林带反转
- KDJ
- RSI

### 套利
- 期现套利
- 跨期套利

### 期权
- 恐慌指数 (VIX)
- 看跌看涨比

## 步骤 4：配置回测

设置以下参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| 策略 | 从模板选择 | 双均线 |
| 标的 | 交易标的 | 000001.SZ |
| 开始日期 | 回测开始日期 | 2024-01-01 |
| 结束日期 | 回测结束日期 | 2024-12-31 |
| 初始资金 | 起始资金 | 100000 |
| 手续费 | 交易佣金比例 | 0.001 |

## 步骤 5：运行回测

点击"开始回测"，等待结果（通常每年数据 < 1 秒）。

## 步骤 6：查看结果

### 分析面板

- **权益曲线** - 组合价值随时间变化
- **回撤曲线** - 最大回撤可视化
- **月度收益热力图** - 月度表现
- **交易统计** - 胜率、盈亏比等

### 关键指标

| 指标 | 说明 |
|------|------|
| 总收益率 | 总体收益率 |
| 年化收益率 | 年化收益 |
| 夏普比率 | 风险调整收益 |
| 最大回撤 | 最大回撤 |
| 胜率 | 盈利交易占比 |
| 盈亏比 | 平均盈利/平均亏损 |
