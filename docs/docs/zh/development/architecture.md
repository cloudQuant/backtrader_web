# 系统架构

## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│                         前端 (Vue 3)                              │
│  策略管理 | 回测分析 | 模拟交易 | 实盘监控 | 投资组合 | 系统设置    │
└───────────────────────────┬─────────────────────────────────────┘
                            │ HTTP/WebSocket
┌───────────────────────────┴─────────────────────────────────────┐
│                        API 网关层 (FastAPI)                      │
├─────────────────────────────────────────────────────────────────┤
│  认证中间件 │ 权限中间件 │ 限流中间件 │ 日志中间件 │ 异常处理      │
└───────────────────────────┬─────────────────────────────────────┘
                            │
┌───────────────────────────┴─────────────────────────────────────┐
│                          服务层 (Services)                        │
├──────────┬──────────┬──────────┬──────────┬──────────┬─────────┤
│ Strategy │ Backtest │  Paper   │  Live    │ Monitor  │  Data   │
│ Service  │ Service  │ Trading  │ Trading  │ Service  │ Service │
└──────────┴──────────┴──────────┴──────────┴──────────┴─────────┘
                            │
┌───────────────────────────┴─────────────────────────────────────┐
│                            数据层 (Data)                          │
├────────────────────┬────────────────────┬───────────────────────┤
│  SQLite/PG/MySQL  │  Backtrader Engine │  策略模板目录          │
│  (用户/策略/回测)  │  (回测/实盘执行)    │  (118 个内置策略)      │
└────────────────────┴────────────────────┴───────────────────────┘
```

## 后端架构

### 分层架构

```
API 层 (app/api/)
    ↓
服务层 (app/services/)
    ↓
数据库层 (app/db/)
    ↓
ORM 模型 (app/models/)
```

### 核心组件

| 组件 | 路径 | 说明 |
|------|------|------|
| API 路由 | `app/api/router.py` | 聚合所有子路由 |
| 认证服务 | `app/services/auth_service.py` | JWT 认证 |
| 回测服务 | `app/services/backtest_service.py` | 回测编排 |
| 策略服务 | `app/services/strategy_service.py` | 策略 CRUD |
| 实盘服务 | `app/services/live_trading_service.py` | 实盘协调 |

## 前端架构

### 组件结构

```
src/
├── api/              # API 客户端函数
├── components/       # 可复用 Vue 组件
├── views/            # 页面组件
├── stores/           # Pinia 状态管理
├── router/          # Vue Router 配置
└── utils/           # 工具函数
```

### 状态管理 (Pinia)

| Store | 说明 |
|-------|------|
| `auth` | 用户认证状态 |
| `strategy` | 策略列表和当前策略 |
| `backtest` | 回测任务和结果 |
| `websocket` | WebSocket 连接管理 |

## 数据库设计

### 核心表

| 表 | 说明 |
|------|------|
| `users` | 用户账户和资料 |
| `strategies` | 用户策略定义 |
| `strategy_versions` | 策略版本历史 |
| `backtest_tasks` | 回测任务记录 |
| `optimization_tasks` | 优化任务记录 |
| `paper_trading_sessions` | 模拟交易会话 |
| `live_trading_sessions` | 实盘交易会话 |

详见 [数据库设计](./database.md)。
