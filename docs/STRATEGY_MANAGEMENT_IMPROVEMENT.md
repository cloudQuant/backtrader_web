# 策略管理功能完善计划

> 参考 TBQuant3 的策略管理功能，完善 backtrader_web 的策略管理界面

## 一、TBQuant3 策略管理功能概述

根据 TBQuant3 官方文档，其策略管理模块包含以下核心功能：

### 1.1 公式管理器 (策略管理核心)

| 功能模块 | 描述 |
|---------|------|
| **新建公式** | 支持新建技术指标、交易策略、用户函数等类型的公式 |
| **打开编辑** | 打开已有公式进行查看和修改 |
| **编译功能** | 编译公式管理列表中选中的用户函数/策略 (Ctrl+F7) |
| **导入导出** | 支持公式的导入导出，便于策略分享和备份 |
| **公式分组** | 分页显示各类公式，通过标签页进行切换 |
| **公式应用设置** | 为策略单元添加/删除公式，调整公式位置 |
| **策略重新加载** | 加载最新公式dll，合并参数设置 |
| **查找替换** | 代码编辑的查找替换功能 |

**公式类型**:
- **技术指标**: 用于技术分析的指标公式，可加载在图表上显示
- **交易策略**: 生成买卖信号的策略公式
- **用户函数**: 可被其他公式调用的函数，返回计算结果
- **公式函数(Defs)**: 在公式内部定义的临时函数

### 1.2 策略单元管理

| 功能模块 | 描述 |
|---------|------|
| **创建策略单元** | 使用交易策略和商品数据创建策略单元 |
| **删除策略单元** | 删除不需要的策略单元 |
| **导入策略单元** | 从外部导入策略单元 |
| **数据源设置** | 批量添加/删除数据源 |
| **优化参数设置** | 设置参数优化范围和目标 |
| **策略应用设置** | 添加/删除公式，调整公式位置 |
| **策略资金曲线** | 查看策略实时权益曲线 |
| **实时编译** | 策略运行时动态编译，无需一次性全编译 |

### 1.3 多策略组合功能

| 功能模块 | 描述 |
|---------|------|
| **多公式组合** | 交易策略可包含多个公式，按顺序串行执行 |
| **持仓共享** | 公式间共享持仓，但信号独立显示 |
| **公式顺序调整** | 可上下移动调整公式执行顺序 |
| **组合优化报告** | 支持同参数组合和多策略单元组合优化 |
| **策略信号单独显示** | 每个公式的交易信号在图表上独立标注 |

**多策略组合说明**:
- 多个公式可以按设置的顺序依次串行执行
- 相当于把每个公式看成是一个函数，在策略内依次执行
- 整个策略共享一个持仓，但每个公式信号单独显示
- 适用于多条件、多策略的组合应用场景

### 1.5 策略研究功能

| 功能模块 | 描述 |
|---------|------|
| **参数优化** | 对策略参数进行优化测试 |
| **策略回测** | 基于历史数据测试策略表现 |
| **策略调试** | 调试策略执行过程 |
| **样本外递进检验** | 验证策略在样本外数据的表现 |
| **优化报告** | 查看优化结果报告 |
| **优化参数复制** | 复制优化参数到策略单元 |
| **组合优化报告** | 同参数组合/多策略单元组合优化 |
| **样本外检验** | 优化完成后进行样本外数据验证 |

**策略研究界面特点**:
- 创建策略单元后可进行深入测试
- 支持参数优化设置（范围、步长、优化目标）
- 优化任务管理（运行、停止、查看结果）
- 样本外递进检验验证策略稳定性
- 优化参数可批量复制到策略单元

### 1.6 策略交易功能

| 功能模块 | 描述 |
|---------|------|
| **实时资金曲线** | 查看策略实时资金状态 |
| **策略开关** | 根据资金曲线管理策略交易开关 |
| **交易单元设置** | 设置交易相关的参数 |
| **策略应用设置** | 添加/删除公式、调整公式顺序 |
| **策略重新加载** | 加载最新公式，合并参数设置 |
| **信号标注** | 在K线图上标注买卖信号 |

**策略交易界面特点**:
- 选中策略交易单元可查看策略资金曲线
- 右键菜单提供策略应用设置选项
- 支持多公式组合，按顺序执行
- 公式间持仓共享，信号独立显示
- 可通过资金曲线管理策略开关

## 八、待完善项补充

基于 TBQuant 界面分析，补充以下待完善项:

| 功能模块 | 缺陷描述 | 优先级 |
