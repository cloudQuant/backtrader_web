## 自定义快捷键

### 如何修改快捷键

1. 进入 **系统设置** → **快捷键**
2. 点击要修改的快捷键
3. 按下新的组合键
4. 点击 **保存**

### 如何添加自定义快捷键

开发者可以通过修改 `src/composables/useKeyboardShortcuts.ts` 添加新快捷键:

```typescript
registerShortcut({
  key: 'm',
  ctrlKey: true,
  description: '切换迷你模式',
  action: () => toggleMiniMode()
})
```

## 快捷键最佳实践

### 推荐使用场景

1. **高频操作**
   - 运行回测: `Ctrl/Cmd + Enter`
   - 保存策略: `Ctrl/Cmd + S`
   - 导航: 数字键 `1-8`

2. **专业用户**
   - 代码编辑: Monaco Editor 快捷键
   - 快速导航: 单键导航

3. **效率提升**
   - 减少鼠标点击
   - 快速切换视图
   - 批量操作

### 注意事项

1. **输入焦点**
   - 在输入框中时，大部分快捷键会被禁用
   - 只有 `Esc` 在输入框中仍然有效

2. **冲突处理**
   - 应用快捷键优先于浏览器快捷键
   - 某些浏览器快捷键无法被覆盖

3. **辅助功能**
   - 快捷键提示显示在按钮悬停时
   - 按 `?` 随时查看所有快捷键

## 平台差异

### Windows/Linux

- 使用 `Ctrl` 键作为主要修饰键
- 示例: `Ctrl + S`

### macOS

- 使用 `Cmd` (⌘) 键作为主要修饰键
- 示例: `Cmd + S`
- 某些功能使用 `Option` (⌥) 键

## 常见问题

### Q: 快捷键不工作？

**A:** 可能原因:
1. 输入框获得焦点 - 点击空白区域
2. 浏览器扩展冲突 - 禁用相关扩展
3. 快捷键被系统占用 - 修改系统快捷键

### Q: 如何临时禁用快捷键？

**A:** 
- 在输入框中编辑时自动禁用
- 或按住 `Shift` 键可临时绕过某些快捷键

### Q: 可以导出/导入快捷键配置吗？

**A:** 
- 目前快捷键配置存储在浏览器本地
- 计划在未来版本支持导入/导出
