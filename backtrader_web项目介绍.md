# Backtrader Web：开源量化交易全栈平台

## 写在前面

在量化交易领域，Backtrader 作为 Python 生态中最受欢迎的回测框架之一，以其灵活性和强大的功能深受开发者喜爱。然而，传统的 Backtrader 使用方式需要编写大量代码，缺乏可视化界面，对于策略管理、版本控制、参数优化等工程化需求支持不足。

**Backtrader Web** 正是为了解决这些痛点而生——一个基于 Backtrader 的现代化量化交易管理平台，将专业的回测引擎与现代 Web 技术完美结合，为量化交易者提供从策略开发、回测分析到模拟交易、实盘监控的全流程解决方案。

## 项目亮点

### 🚀 开箱即用，5分钟完成首次回测

无需复杂配置，克隆项目后简单几步即可启动：
- 后端基于 FastAPI，自动生成 API 文档
- 前端基于 Vue 3 + Element Plus，现代化 UI 体验
- 默认使用 SQLite，零配置启动（支持 PostgreSQL/MySQL 切换）

- **ECharts K线图**：支持多周期切换、技术指标叠加

## 核心功能详解

### 1. 策略管理

- **策略库**：118 个内置策略模板，包括：
  - 趋势跟踪：双均线、MACD、海龟交易法
  - 反转策略：布林带反转、KDJ、RSI
  - 套利策略：期现套利、跨期套利
  - 期权策略：恐慌指数（VIX）、看跌看涨比
  - 高频策略：R-Breaker、Dual Thrust、菲阿里四价

- **策略编辑器**：
  - 语法高亮
  - 代码自动补全
  - 实时语法检查
  - 参数可视化配置

### 2. 回测分析

**标准化指标**（基于 fincore）：
- 收益指标：总收益率、年化收益率、超额收益
- 风险指标：最大回撤、波动率、下行风险
- 风险调整收益：夏普比率、索提诺比率、卡玛比率
- 交易统计：胜率、盈亏比、平均持仓时间

**可视化图表**：
- 权益曲线 + 回撤曲线
- 月度收益热力图
- 收益分布直方图
- 交易记录时间轴
- 持仓分析饼图

### 3. 参数优化

- **优化算法**：网格搜索（Grid Search）
- **并行计算**：多进程加速
- **结果分析**：
  - 参数热力图（2D）
  - 参数曲面图（3D）
  - 最优参数组合推荐
  - 参数敏感性分析

### 4. 模拟交易

- **账户管理**：多账户支持，独立资金管理
- **订单系统**：市价单、限价单、止损单、止盈单
- **持仓管理**：实时持仓、历史持仓、盈亏统计
- **交易记录**：完整的交易流水，支持导出

### 5. 实盘交易

- **券商对接**：支持多家券商 API
- **实例管理**：多策略并行运行
- **风险控制**：
  - 单笔交易限额
  - 日内交易次数限制
  - 最大持仓比例
  - 止损止盈自动执行

### 6. 监控告警

- **告警规则**：
  - 收益率告警（盈利/亏损阈值）
  - 回撤告警
  - 持仓告警
  - 订单异常告警

- **通知渠道**：
  - 邮件通知
  - 短信通知
  - Webhook（钉钉、企业微信、Slack）

### 7. 投资组合

- **组合管理**：多策略组合配置
- **权重分配**：固定权重、动态再平衡
- **组合分析**：
  - 组合收益曲线
  - 策略贡献度分析
  - 相关性矩阵
  - 风险分散效果

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 20+

### 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/cloudQuant/backtrader_web.git
cd backtrader_web

# 2. 后端安装
cd src/backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -e ".[dev,backtrader]"
cp .env.example .env

# 3. 启动后端
uvicorn app.main:app --reload --port 8001

# 4. 前端安装（新终端）
cd src/frontend
npm install
npm run dev
```

### 访问地址

- **前端界面**：http://localhost:3000
- **API 文档**：http://localhost:8001/docs
- **WebSocket**：ws://localhost:8001/ws

### 5分钟完成首次回测

1. **注册账号**：访问前端页面，注册新用户
2. **选择策略**：从 118 个内置策略中选择一个（如"双均线策略"）
3. **配置参数**：设置回测时间、初始资金、手续费等
4. **运行回测**：点击"开始回测"，等待结果
5. **查看分析**：查看权益曲线、回撤曲线、交易记录等
